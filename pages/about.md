## Почему именно специализированные гайды

Большинство курсов английского устроены для абстрактного «среднего ученика».
Это не работает для практиков:

- IT-разработчику не нужна лексика про «my favourite holiday» —
  ему нужны фразы для стендапа и system design.
- Врачу не нужны диалоги в кафе — ему нужны скрипты для приёма пациента
  по Calgary-Cambridge.
- Релоканту не нужны грамматические упражнения — ему нужно завтра
  объяснить лендлорду про source of funds.

Я делаю гайды, которые решают **конкретные** боли **конкретной** аудитории.
Без воды, без общей теории, без «выучите 500 неправильных глаголов».

## Принципы

1. **Узкая специализация вместо общих курсов.** Один гайд = одна задача.
2. **Применимость с первого дня.** Не теория, а готовые скрипты и шаблоны.
3. **Свой темп, без подписки.** Один платёж — материал ваш навсегда.
4. **Регулярные обновления.** Особенно для гайдов с быстро меняющимся
   контекстом (Legal, Logistics, Relocation).
5. **14 дней на возврат без объяснения причин.** Если не подошло — деньги
   возвращаются.
